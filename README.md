## Installation

```bash
npm install @agent-infra/agent-deploy
```

## Quick Start


```go
import "github.com/yksanjo/agent-infra-agent-deploy"

func main() {
    instance := AgentDeploy.New(AgentDeploy.Config{})
    instance.Initialize()
    result := instance.Execute(input)
}
```


## API Reference

### `AgentDeploy`

Main class for agent deploy functionality.

#### Methods

- `initialize()` - Initialize the component
- `execute(input)` - Execute main logic  
- `configure(config)` - Update configuration
